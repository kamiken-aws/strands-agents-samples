# Strands Agent サンプル



| 例 ID | エージェント                                                               | 紹介される機能                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ---------- | ------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1          | [レストランアシスタント](./01-restaurant-assistant/)                  | 顧客インタラクションを効率化・向上させるレストランアシスタント。AI アシスタントにより、顧客は複数のパートナーレストランでテーブルを予約したり、メニューを問い合わせたり、新規予約を作成したり、既存の予約の詳細を取得したり、予約をキャンセルしたりすることが、自然な会話インターフェース（Web、モバイル、または音声）を通じてシームレスに行えます。この自動化により顧客満足度が向上し、スタッフの作業負荷が軽減され、予約効率が向上し、レストランには顧客の好みや予約トレンドに関する貴重な洞察が提供されます。 |
| 2          | [スクラムマスターアシスタント](./02-scrum-master-assistant/)              | JIRA 用のエキスパートアジャイル開発アシスタント。会議メモを実行可能なタスクに分解することに特化しています。                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 3          | [AWS アシスタント MCP](./03-aws-assistant-mcp/)                        | Model Control Protocol（MCP）を使用した AWS に焦点を当てたアシスタント                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 4          | [スタートアップアドバイザー MCP](./04-startup-advisor-mcp/)                    | Model Control Protocol（MCP）を使用したスタートアップアドバイザリーエージェント                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| 5          | [パーソナルアシスタント（マルチエージェントシステム）](./05-personal-assistant/) | カレンダー管理、Perplexity MCP Server 経由の Web 検索、インテリジェントなマルチエージェントアーキテクチャによる開発支援を統合した包括的な生産性アシスタント。                                                                                                                                                                                                                                                                                                                                                                       |
| 6          | [コードアシスタント](./06-code-assistant/)                              | コードアシスタントエージェントは単なるツールではなく、開発ワークフローを強化するために設計された AI 搭載の個人的なコーディングコンパニオンです。                  |
| 7          | [フィンテックアシスタント（WhatsApp 統合）](./07-whatsapp-fintech-sample/) | ツールを使用して顧客サポートを調整する WhatsApp 上のフィンテックアシスタントボット。曜日に基づいて毎日のプロモーションを読み込みます。別のエージェントはクレジットカード操作の処理を担当し、過去 X 日間の偽のトランザクションを読み込んだり、カードの支払いをスケジュールしたりできます。                   |
| 8          | [データウェアハウスオプティマイザー（SQLite）](./08-data-warehouse-optimizer/) | SQLite を使用したデータウェアハウスクエリオプティマイザーをシミュレートするマルチエージェントシステム。Analyzer、Rewriter、Validator エージェントが協力して SQL クエリプラン（`EXPLAIN QUERY PLAN`）を分析し、最適化を提案し、改善を検証します。AWS Bedrock の Claude 3 Haiku を使用し、OpenTelemetry で可観測性をログに記録。分析と検証結果を含む最終的な JSON レポートを生成します。                                                                                                                    |
| 9          | [ファイナンスアシスタント Swarm エージェント協調](./09-finance-assistant-swarm-agent/) | ファイナンスアシスタント Swarm エージェント協調は、単一の株式クエリから包括的な株式調査レポートを自律的に生成するように設計されたモジュラーなマルチエージェントシステムです。Strands SDK を使用して構築され、Amazon Bedrock を搭載したこのアシスタントは、ティッカー解決、会社プロファイリング、価格分析、財務健全性評価、センチメント分析を含む個別の財務調査タスクを担当する専門エージェントの協調スウォームを調整します。                   |
| 10         | [RAG と画像生成を備えたメールアシスタント](./10-multi-modal-email-assistant-agent/) | マルチモーダルメールアシスタントは、エンタープライズコミュニケーションのためのエージェント協調の力を示し、マーケティング、レポート、顧客エンゲージメントなどの領域でプロフェッショナルなコンテンツ作成を自動化するスケーラブルなフレームワークを提供します。             |
| 11         | [パーソナルファイナンスアシスタントワークショップ](./11-personal-finance-assistant/) | 予算分析エージェントの構築を通じて Strands Agents の基礎を教える 3 部構成のワークショップ。エージェント作成、カスタムツール、Amazon Bedrock Guardrails 統合、メモリシステム、個人財務管理のためのマルチエージェントオーケストレーションをカバーします。基礎、メモリ統合、高度なオーケストレーションパターンのハンズオンラボを含みます。 |
| 12        | [医療文書処理アシスタント](./12-medical-document-processing-assistant/) | 医療文書処理アシスタントは、PDF や画像などのさまざまな文書形式から医療情報を抽出、分析、強化するために設計された AI 搭載ツールです。このアシスタントは、臨床メモ、病理レポート、退院サマリー、その他の医療文書を処理して、標準化された医療コーディングで構造化されたデータを提供することに特化しています。             |
| 13        | [AWS インフラストラクチャ監査アシスタント](./13-aws-audit-assistant/) | AWS 監査アシスタントは、AWS リソースのベストプラクティスへの準拠を保証するための AI 搭載パートナーです。セキュリティと効率性の改善のためのインテリジェントな洞察と推奨事項を提供します。     |
| 14        | [リサーチエージェント](./14-research-agent/) | ホットリロードツール作成、マルチエージェントオーケストレーション、セッション間の永続的学習、分散インテリジェンスを備えた自己改善 AI システムを紹介する自律リサーチエージェント。調整プリミティブがリサーチタイムスケーリングのための認知ツールになるメタ認知アーキテクチャを示します。 |
| 15        | [カスタムオーケストレーション航空アシスタント](./15-custom-orchestration-airline-assistant/) | Strands GraphBuilder API を使用したカスタムマルチエージェントオーケストレーションパターン。4 つの異なる調整戦略（ReAct、REWOO、REWOO-ReAct ハイブリッド、Reflexion）を、明示的なエージェントワークフロー、非同期実行フロー、航空会社の顧客サービスシナリオにおける複雑なタスク自動化のための完全な可観測性で実証します。 |
| 16         | [Lambda エラー分析エージェント](./16-lambda-error-analysis-agent/) | AI 搭載の根本原因分析によるインテリジェントな Lambda エラー診断。イベント駆動アーキテクチャ、マルチツール Strands エージェント（ソースコード、ログ、Knowledge Base）、信頼度スコアリング、Bedrock Knowledge Base 統合を備えた完全な CDK インフラストラクチャを特徴とします。エラー分析を自動化し、Lambda 障害に対する実用的な推奨事項を提供します。 |
| 17         | [GenAI 搭載の資産・金融アドバイザリーツール](./17-genai-powered-financial-advisor-tools/) | Strands Agents、Amazon Bedrock、MCP を使用して構築された包括的な GenAI 搭載の資産・金融アドバイザリーツール。金融アドバイザー向けのクライアントミーティング分析、アドバイザーフォローアップ項目分析、クライアントポートフォリオ分析、市場調査、レポート生成を自動化するように設計されています。 |
| 18         | [暗号通貨決済を備えた自律 AI 広告エージェント](./18-ai-ads-generation-agent-with-crypto-payments/) | 完全な広告キャンペーンを作成し、X402 決済プロトコルを介して暗号通貨を使用してプレミアムサービスに対して自律的に支払いを行う AI エージェント。天気に応じた広告生成、AI 画像生成、Base Sepolia テストネット上の USDC 決済、完全な HTML キャンペーン出力による経済的エージェンシーを実証します。Strands Agents を Coinbase AgentKit と統合します。 |
